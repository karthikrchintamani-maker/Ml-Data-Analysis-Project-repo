# 🤖 ML Data Analysis Projects – EPCET

## 📌 Overview

This repository contains a collection of Python-based data analysis projects developed as part of an AI/ML assignment. The projects use **Pandas**, **Matplotlib**, and **Seaborn** to analyze different Indian datasets and visualize useful insights.

---

## 📊 Included Projects

### 🦠 1. COVID-19 Data Analysis (`data.py`)

**Dataset:** `covid_19_india.csv`

**Analysis**
- Filter COVID-19 records by date
- Find states with the highest number of deaths
- Analyze state-wise death trends
- Display results using bar charts

---

### 🔥 2. LPG Statewise Analysis (`lpg.py`)

**Datasets**
- `India_lpg_dataset.csv`
- `LPG_Statewise_Migration_Dataset_500_Rows.csv`

**Analysis**
- Compare LPG consumption across states
- Analyze LPG shortage by state
- Compare migration statistics
- Generate separate visualization charts

---

### 🏥 3. Cancer Data Analysis (`cancer.py`)

**Dataset:** `Cancer_Statewise_Dataset_500_Rows.xlsx`

**Analysis**
- State-wise cured cancer cases
- State-wise cancer deaths
- Compare cured cases with deaths
- Display grouped bar charts

---

### 🚦 4. Traffic Violations Analysis (`driving.py`)

**Dataset:** `Indian_Traffic_Violations.csv`

**Analysis**
- Fine amount by violation type
- State-wise traffic violations
- Violation distribution across states
- Seaborn visualization charts

---

## 📂 Datasets

| Dataset | Description |
|----------|-------------|

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/santhosh-d-projects/ML-DATA-ANALYSIS-EPCET.git
cd ML-DATA-ANALYSIS-EPCET
```

### Install Required Libraries

```bash
pip install -r requirements.txt
```

### Execute the Programs

```bash
python "Python Codes/data.py"
python "Python Codes/lpg.py"
python "Python Codes/cancer.py"
python "Python Codes/driving.py"
```
